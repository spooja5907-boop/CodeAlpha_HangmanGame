# 🎮 Hangman Game

## 📌 Description
This is a simple text-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Randomly selects a word
- Letter-by-letter guessing
- Maximum of 6 wrong attempts
- Displays the guessed letters
- Win/Loss message

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Download the project.
2. Open the file in Python IDLE or any Python IDE.
3. Run the program.
4. Guess the letters and enjoy the game!
